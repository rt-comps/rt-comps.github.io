A graphical menu item for order form product menu.
Created during form initialisation from rt-itemdata elements

Attributes:
- Required:
    - id
        - Must be unique for this rt-orderform
- Optional:
    - bgimg
        - The filename of a background image.  It is assumed all images are in a single location

Events:
- Dispatched:
    - initmenu
        - Sends component ID when component clicked
- Listened: None

Dependancies: None

Notes: None
